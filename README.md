## Run this project:

- Make sure you have Expo Go on your phone to scan the QR or enter the exp local domain 
- Node, NPM and Yarn is required 
- Get the `.env` file from this link (Supabase free-tier API limits): https://drive.google.com/drive/folders/1ZMeiaHJqLLWzAlLF5opHzaR3-4hpcUjA?usp=sharing

Clone the repo 
```bash
cd Twins 
yarn install
npm start
```
