Overview
- Semi-closed social app to find new friends via a Big Five personality quiz.
- This repo contains an Expo-style RN scaffold with TypeScript, theming, navigation, questionnaire, scoring, and a radar chart placeholder.

Getting Started
- Install dependencies: npm install (requires Expo SDK 51+)
- Run dev server: npm start
- iOS/Android: use Expo Go or run native with expo run:ios / run:android

Structure
- App.tsx: Theme provider + Navigation container
- navigation/AppNavigator.tsx: Stack with Registration, Questionnaire, Results
- components/: Screens + common UI + charts/RadarChart
- context/ThemeContext.tsx + themes/: Dynamic dark theme palette (RGB triplets)
- data/questions.ts: IPIP-50 item bank
- services/profileAnalyzer.ts: Scoring utilities (+/- keyed, normalization)
- hooks/useTensorflowModel.ts: Placeholder hook for future TF integration

Notes
- Fonts (Inter) and model assets are not committed. Add under assets/fonts and assets/ml.
- Firebase/GCP auth + storage are planned for a later phase.
- UI prefers dark mode and fade transitions per design doc.
