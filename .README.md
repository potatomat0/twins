Overview
- Semi-closed social app to find new friends via a Big Five personality quiz.
- Built with Expo SDK 53 (React Native 0.79.5, React 19).

Prerequisites (CLI-driven)
- OS: macOS 13+, Windows 11, or Ubuntu 22.04+.
- Node.js: 20.x LTS recommended (check: node -v).
- npm: 10.x+ (check: npm -v).
- Expo CLI (bundled via npx) and EAS CLI for builds:
  - npm i -g eas-cli
- For Android local runs: Android Studio + SDK Platform 34, USB debugging enabled.
- For local native builds (optional): JDK 17 and Android SDK/NDK properly on PATH.

Install & Run (Dev)
1) Clone and install
   - npm install
   - npx expo install --check (accept suggested fixes)
2) Start Metro + Expo
   - npm start
   - Press a (Android emulator) or scan QR in Expo Go
3) Troubleshooting
   - npx expo-doctor
   - npx expo start -c (clear cache)

EAS Build (APK/AAB)
- Login and configure
  - eas login
  - eas build:configure
- Universal APK (sideload/testing)
  - eas build -p android --profile preview
- Play Store bundle (AAB)
  - eas build -p android --profile production
- Download the artifact from the URL printed after build finishes.

Project Structure (Brief)
- App.tsx: Theme provider + Navigation container
- navigation/AppNavigator.tsx: Stack (Registration, Questionnaire, Results)
- components/
  - common/: Button, Card, Dropdown, LoadingSpinner
  - charts/RadarChart.tsx: Interactive radar chart + info overlay
  - RegistrationScreen.tsx, QuestionnaireScreen.tsx, ResultsScreen.tsx
- context/ThemeContext.tsx + themes/index.ts: Dynamic dark theme (RGB triplets)
- data/questions.ts: IPIP-50 items
- data/factors.ts: Factor explanations (from project document)
- services/profileAnalyzer.ts: IPIP scoring + normalization
- hooks/useTensorflowModel.ts: Placeholder for future TF integration
- app.json: App config (Android package, versionCode, minSdk)
- eas.json: EAS build profiles (preview APK, production AAB)
